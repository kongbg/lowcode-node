# lowcode-node



### 克隆代码
    git clone https://github.com/kongbg/lowcode-node.git

### 进入项目目录
    cd  /lowcode-node

### 安装依赖
    npm install

### 本地启动(先到.env文件中配置数据库)
    npm run start

### 构建docker(前提是本机安装了docker)
    docker build -t lowcode-node .
